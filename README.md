# My Frontend Development Portfolio

This repository contains three static website projects focused on beginner-level frontend development. Each project demonstrates different layout techniques and UI designs.

📂 Projects Overview

  [IT Learning Platform](https://sam-hana.github.io/myWeb/day7/)
An IT education learning website showcasing clean structural layout and UI design.

  [Online Shopping Website](https://sam-hana.github.io/myWeb/day9_10/)
An e-commerce website layout practicing fundamental frontend skills.

  [Music Platform Web App](https://sam-hana.github.io/myWeb/kuwo/)
A simple music platform webpage featuring responsive design to maintain proper proportions across different devices.


Tech Stack: HTML5, CSS3
